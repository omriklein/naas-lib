# naas-lib

**No-as-a-Service** — get a random, creative excuse to say no.

## Installation

```bash
npm install naas-lib
```

## Usage

### As a library

```js
const getNo = require('naas-lib');

console.log(getNo()); // "I'd love to, but my houseplants need me."
```

### As a CLI

```bash
npx naas
```

Or install globally:

```bash
npm install -g naas-lib
naas
```
